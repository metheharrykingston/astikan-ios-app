Use Google sign-in or phone verification to access health services, lab tests, pharmacy, teleconsultation and wellness support from one secure app.

Open Web App

Android: tap Install App. If the prompt does not appear, open Chrome menu and choose Add to Home screen.

Desktop: use the Chrome/Edge install icon in the address bar, or open the web app directly.
